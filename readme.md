<h2 id="projeto">PROJETO</h2>

Projeto voluntário desenvolvido para orientar e facilitar o acesso de candidatos às oportunidades de emprego e estágio da Nexa pelo portal da Gupy. O objetivo foi reduzir barreiras no processo de candidatura — especialmente para o público operacional — entregando um guia simples, didático e acessível sobre a empresa, as vagas e o passo a passo de inscrição.

🌐 [Acesse o site](https://portalnexa.vercel.app/)

<h2 id="funcionalidades">FUNCIONALIDADES</h2>

- **Apresentação da Nexa** — explicação clara sobre a empresa e suas oportunidades
- **Link direto para o portal** — acesso imediato às vagas via Gupy
- **Passo a passo** — seções expansíveis com guia didático de candidatura
- **FAQ** — respostas para as dúvidas mais frequentes dos candidatos
- **Tema claro/escuro** — alternância via toggle button conforme preferência do usuário
- **Layout responsivo** — interface adaptada para todos os dispositivos
